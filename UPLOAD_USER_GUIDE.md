# File Upload User Guide

## Quick Start

### How to Upload Files

#### Method 1: Click the Attachment Button
1. Look for the **paperclip icon** (ğŸ“) in the chat input area
2. Click it to open the file picker
3. Select one or more files from your computer
4. Files will appear as thumbnails above the input field
5. Type your message and press Enter to send

#### Method 2: Drag and Drop
1. Select files from your file explorer/finder
2. Drag them directly into the **chat input box**
3. Files will appear as thumbnails
4. Type your message and press Enter to send

#### Method 3: Copy-Paste Images
1. Copy an image to your clipboard
2. Click in the message input field
3. Press Ctrl+V (Windows) or Cmd+V (Mac)
4. Image will be attached automatically

### Troubleshooting

#### Upload Fails or Times Out
- Check your internet connection
- Try uploading a smaller file first
- Wait a few seconds and retry
- Try a different file format

#### File Not Recognized
- Check that your file format is supported
- Ensure file is not corrupted
- Try renaming the file with correct extension

#### Files Don't Appear
- Check the console (F12) for error messages
- Refresh the page and try again
- Clear browser cache and retry

## What Happens After Upload

1. **File Attached** - Files appear as thumbnails in the input area
2. **Message Sent** - When you press Enter, files upload to the server
3. **Processing** - The AI analyzes the file content
4. **Response** - The AI provides insights based on file content

## Removing Files

**Before Sending:**
- Hover over a file thumbnail
- Click the X button that appears
- File will be removed from the queue

**After Sending:**
- Files are attached to that message
- Cannot be removed after sent
- Upload again to replace

## Common Questions

**Q: Can I upload multiple files at once?**
A: Yes! Select up to 10 files in the file picker, or drag multiple files at once.

**Q: What if my file is larger than 50MB?**
A: Compress the file or split it into smaller pieces, then upload separately.

**Q: Can I edit files after uploading?**
A: No, files are sent as-is. Modify locally and re-upload if needed.

**Q: Are my files saved?**
A: Files are processed and may be retained for model training (depends on privacy settings). Check your privacy policy.

**Q: Can I undo a file upload?**
A: Before sending: Remove it with the X button. After sending: It's part of the message and cannot be removed.

**Q: What's the fastest upload speed?**
A: Depends on your internet connection. Typically 1-10 MB/s on broadband.
